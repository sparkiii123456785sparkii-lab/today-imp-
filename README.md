# SparkleBots Internship Program

## Run the project

1. Open the folder in VS Code:
   - `File` -> `Open Folder...`
   - Select `c:\Users\Lenovo\Desktop\10th intership`

2. Install dependencies:
   ```powershell
   npm install
   ```

3. Start the Next.js server:
   ```powershell
   npm run dev
   ```

4. Open the website:
   - `http://localhost:3000`

## Go Live shortcut

If you click `Go Live`, VS Code will open `index.html` in the root and automatically redirect to `http://localhost:3000/`.

> Make sure the dev server is running first with `npm run dev`.
